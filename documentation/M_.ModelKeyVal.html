Class: ModelKeyVal

M_.ModelKeyVal()

new ModelKeyVal()

Use a standard model key / value. No params.
Example
// Very useful with combobox
var combo = new M_.Form.Combobox({
	name: 'of_typecook',
	label: "Type de cuisine",
	container: $("#here"),
	store: new M_.Store({
		controller: this,
		model: M_.ModelKeyVal,
		currentSort: ["key", 1],
		rows: [{key:1, val:"Cuisine américaine"}, {key:2, val:"Cuisine niçoise"}]
	})
}) ;
